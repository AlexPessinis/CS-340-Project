# CS-340-Project
Full python / MongoDB CRUD module with interactive website developed in my CS-340 class.
## How do you write programs that are maintainable, readable, and adaptable?
Writing maintainable and adaptable programs depends many things, such as clear structure, consistent naming, modular design, and documentation that explains intent rather than just behavior. In the CRUD Python module, separating database operations into a dedicated class made the code easier to reuse and update. Each method handled a single responsibility, which reduced duplication and made debugging more predictable. This structure also made it simple to connect the module to the dashboard in Project Two, because the dashboard only needed to call well‑defined functions rather than rewrite database logic. The same module could be reused in future applications, such as automated reporting tools, API endpoints, or expanded dashboards, without rewriting the underlying database operations.

## How do you approach a problem as a computer scientist?
Approaching a problem involves breaking it into smaller, testable components, identifying constraints, and designing a solution that can evolve as requirements change. For the Grazioso Salvare project, this meant understanding the data first, then designing queries that matched the client’s rescue criteria, and finally building a dashboard that presented the results clearly. Compared to earlier coursework, this project required more emphasis on real qorld requirements, modularity, which I quite enjoyed. Future database projects would benefit from the same strategies: gathering requirements early, designing schemas and queries around real use cases, and building components that can be reused or extended as client needs grow.

## What do computer scientists do, and why does it matter?
Computer scientists design systems that transform data into useful, actionable information. Their work matters because organizations rely on accurate, efficient tools to make decisions, automate processes, and understand complex information. In a project like this, the CRUD module and dashboard help a fictional company such as "Grazioso Salvare" quickly identify animals suited for specialized rescue roles. This improves their efficiency, reduces manual searching, and supports better outcomes for both the organization and the animals. The ability to turn raw data into meaningful insights is why computer science has such a significant impact across industries.
